Main type of document on the web:

HTML

Types of files on the web:

  • Plain text
  • HTML
  • Images
  • Videos
  • Music

Components of the web:

  1. HTTP - web protocol
  2. Servers - host web files
  3. Internet - worldwide computer network
  4. Browser - program to display web files

HTML Tags:

  • Bold - indicated by <b>

    • This text is bold

  • Italics - indicated by <em>

    • This text is italic

  • Paragraph - indicated by a <p>

    • This is a paragraph

  • Span - in-line tag that encloses content, indicated by a <span>

    • This is a span

  • Div - block tag that encloses content, indicated by a <div>
    • This is a div

Creating a Structured Document

  1. Websites have a tree-like structure (DOM)
  2. HTML controls the structure of a page
  3. HTML has a specific syntax and rules
  4. Basic word in HTML is a tag

Styling an HTML page

  1. CSS controls the style of a page (appearance)
  2. changes how elements appear on the page
  3. Style elements CSS can change:
    • Positioning
    • Font and Text Style
    • Background and text color
    • ...and many other elements
